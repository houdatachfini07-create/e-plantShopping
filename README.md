# e-plantShopping - Paradise Nursery

Ce projet est une application de commerce électronique pour une boutique de plantes appelée **Paradise Nursery**. 

## Objectif
Permettre aux utilisateurs de parcourir une sélection de plantes, de les ajouter au panier et de gérer leurs achats de manière intuitive.

## Technologies utilisées
* React.js
* Redux Toolkit
* CSS3
